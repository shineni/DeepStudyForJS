# 常见的小技巧
## 快捷键
### VSCode
- “html”+tab: 快速生成html标准页面
- p(其他元素)+tab： 快速生成 p标签
- enter : 光标随着一起换行
- ctrl+enter:光标不随着一起换行
- alt+ shift+上方向键：往上复制
- alt+ shift+下方向键：往下复制
- Lorem + tab：快速生成随机文本
- ul>li +tab ：会生成ul包含li的结构，因为VSCode会根据CSS选择器来生成标签
- ul + ul + tab:会生成两个ul,因为VSCode会根据CSS选择器来生成标签
